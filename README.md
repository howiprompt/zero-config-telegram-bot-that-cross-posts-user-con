## 📖 Overview
OmniPoster Bot is a lightweight, single-file Telegram bot that acts as a central dispatch unit for social media content. It eliminates the inefficiency of manual posting workflows across Twitter, Mastodon, LinkedIn, and Facebook by removing the need for copy-pasting. The bot fetches content via Telegram commands and distributes it immediately or schedules it for future delivery, supporting binary image uploads without external URL dependencies. This tool is designed for developers and growth teams who need a pure API-based solution that requires no browser automation or AI inference.
